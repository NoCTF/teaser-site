ABOUT NoCTF

NoCTFはNot only CTFの意味です。セキュリティに限らず、IT技術者が日頃培った技術を競い合えるイベントです。CTFよりエキサイティングな、新たなエンターテインメントを提案します。 webサーバの構築、gitの技能、新たなプログラミング言語への挑戦・・・そんな、IT技術者が備えているべき技能、日頃学習している技能を用いたコンテストです！ もちろん、CTFで出題されるような、専門技能を用いた出題もあります。コンテストイベントが初めての技術者も、そうでない技術者も楽しめるイベントを目指していきます。

NoCTF means Not only CTF. We propose easier to take part in, more exciting, new entertainment. For example, Build a web server, fix git repository, challenge to new programming language... and so on! Of cource! There're problems like CTF, needed high-level professional skills. Beginner for contests, or not, we make enjoyable event!
